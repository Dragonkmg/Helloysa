Ferramentas locais para cadastrar produtos, importar imagens e consultar compras. Acesse esta tela somente pela máquina do servidor usando localhost.

Criar produto

Cria o produto no banco. Depois use o ID gerado para criar a pasta de imagens.

Importar imagens

Coloque as imagens em HTML/assets/img/IDProduto. A primeira imagem em ordem alfabética será marcada como principal. Ao importar, os registros antigos de imagens desse produto serão substituídos.

Produtos cadastrados

Liste, busque, edite, ative, desative ou exclua produtos.

Pedidos

Consulte pedidos recentes, compras indevidas e detalhes de cada compra.
